Ft base64

A python module replacing the native base64.encodestring()

What

Superfast replacement for 2 functions form the python base64 module. Provided are functions that can replace base64.b64encode() and base64.encodestring().

Why?

Well, agree that normally running a function such as this in 0.00001 or 0.000001 seconds does not really make a difference, but if you have some time sensitive and/or massive processing going on, 10x improvement can mean a lot

How

... to install

$ git clone git@github.com:KillerX/ft_base64.git
$ sudo python setyp.py install

... to use

The functions are almost drop in replacement.

Otherwise here is an example:

import ft
import base64

"""ft.base64 behaves identically to base64.b64encode"""
shortString = "The quick brown fox jumps over the lazy dog!!"
a = base64.b64encode(shortString)
b = ft.base64(shortString)

"""ft.base64string behaves similarly to base64.encodestring. 
See lines below for the difference"""
ft.base64string("FOO")
base64.encodestring("BAR").replace("\n", "\r\n").strip()
